이자용

안녕하세요 이자용입니다. 저는 86년생으로 경영학/시각디자인학과를 졸업하고 현재 프론트엔드 개발자로 일하고 있습니다. 웹퍼블리셔 경력은 9년이며, 최근에 시작한 프론트엔드 업무 경력은 1년입니다.
마크업개발자들은 현재 선택의 기로에 서 있다고 생각합니다.
디자이너 성향의 마크업개발자 혹은 프론트엔드 개발자로 영역을 넓히지 않으면 이 시장에서 수명은 길지 않을 거라고 생각합니다.
저는 저의 가치를 높이고 나아가 제가 원하는 삶을 이루기 위해 끊임없이 배우는 중입니다.
앞으로는 프론트엔드와 react 전문가로 제 자신을 브랜딩하고 싶습니다. node.js와 python, mongo db로 간단한 웹사이트를 만들어본 경험이 있고, react를 활용하여 컴포넌트를 나누어 sass를 통해 자유자재로 스타일링이 가능합니다.
또 useState와 context API, redux / redux-saga로 상태관리를 하고 firebase를 통해 직접 간단한 db연동해보기도 하고, 마지막으로 aws를 통해 react빌드된 프로젝트를 s3환경에서 배포하고 도메인 연결이 가능합니다.
최근에는 nextJs 를 이용한 개발경험이 있고 redux-saga 와 redux-toolkit을 이용해 상태관리 하고있습니다. 숙련도 측면에서 아직 프론트엔드 개발자로서는 어설프다고 볼 수 있겠지만 저는 수년 간 마크업개발자로 경력을 인정받은 경험을 바탕으로 포기하지 않을 것입니다.
